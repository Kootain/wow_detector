# 引擎核心流程


# 核心模块
## Ingame State
1. 游戏内状态的数据模型定义
    1. Buff
    2. 施法状态
    3. 资源
        1. 法师资源
    4. 冷却
2. 测试用例

## State Mananger
1. 泛化的链式调用
2. 链式调用的合法性检查
3. 泛化的更新检查

## Action
1. 当前动作状态感知
2. channel 动作的 引导时间设定
3. 下一个动作的schedule

## Stratgy
1. apl语法解析
2. 实现condition的执行
3. 实现action的输出

## State Monitor
1. 绑定strategy 关注的state
2. 串联 IngameState & StateManager, 负责更新state
3. 检查变更带来的影响
